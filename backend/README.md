## Description

RESTful API developed with [Nest](https://github.com/nestjs/nest) framework in Typescript for [CookUnity](https://www.cookunity.com/) Engineer Coding Challenge.

## Manual Installation

```bash
$ npm install
```

### Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Or build & run with Docker

```bash
docker-compose up --build
```
